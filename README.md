# Sistema de Gestão de Itens

## 📌 Descrição
Este projeto foi desenvolvido no âmbito da disciplina de Programação com Base de Dados.
O objetivo é implementar um sistema de gestão de itens utilizando exclusivamente SQL.

## 🎯 Funcionalidades
- Cadastro de categorias
- Cadastro de produtos
- Controle de stock
- Registo de entradas e saídas através de stored procedures
- Consultas para relatórios

## 🛠️ Tecnologias Utilizadas
- MySQL
- MySQL Workbench

## 📂 Estrutura do Projeto
database/ → Scripts SQL organizados por etapa  
docs/ → Documentação e diagrama ER  

## 🚀 Como Executar
1. Executar 01_create_database.sql
2. Executar 02_create_tables.sql
3. Executar 03_insert_data.sql
4. Executar 04_procedures.sql
5. Executar 05_queries.sql
